## Les Cinq Commandements de l'Acte 2

### INCIDENT DÉCLENCHEUR ACTE 2

> Après 8 mois de voyage, le *Thulé* arrive en vue de l'Objet. Il est exactement comme prévu — une sphère noire parfaite de 12 km, absorbant presque toute la lumière. Le signal radio s'est arrêté à leur arrivée.

### POINT MÉDIAN (MIDPOINT)

> **Fausse victoire** : Élise décode le mécanisme d'ouverture. Un sas s'ouvre dans la surface de l'Objet. L'équipage peut entrer.

### "ALL IS LOST" MOMENT

> Au centre de l'Objet, Élise découvre la vérité : c'est un **mausolée**. Des milliers de capsules contenant les restes de civilisations intelligentes disparues. L'humanité est peut-être la dernière espèce consciente de la galaxie.

### TURNING POINT ACTE 2

> Les hologrammes révèlent le message : l'Objet a été placé ici il y a 4,7 milliards d'années, quand la Terre n'existait pas encore, en espérant que quelqu'un le trouverait. Nous sommes les **héritiers désignés**.

### CRISE ACTE 2

**Type** : Meilleur Mauvais Choix

> L'Objet contient toute la connaissance des créateurs, mais l'interface nécessite une connexion neuronale directe — risque élevé de mort ou de dommages cérébraux. Élise doit choisir : risquer sa vie pour cette connaissance, ou rentrer avec ce qu'ils ont déjà.

### CLIMAX ACTE 2

> Élise choisit de tenter la connexion. Elle entre dans l'interface alien.

### RÉSOLUTION ACTE 2

> La connexion fonctionne. Élise accède à une fraction de la connaissance. Elle survit, mais transformée.

## SCÈNES DÉTAILLÉES

### Scène 7 : L'Arrivée

**~1 200 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Pont du *Thulé*, approche de l'Objet |
| **Temps** | Décembre 2147 (8 mois après le départ) |
| **POV** | Élise |

**Résumé** : Premier contact visuel avec l'Objet. C'est exactement comme les données le prédisaient — et pourtant, le voir est différent. Une sphère noire parfaite, 12 km de diamètre, qui semble absorber la lumière. Le signal radio s'est arrêté au moment où ils sont arrivés à portée.

**Les Cinq Commandements** :
- **Incident déclencheur** : L'Objet apparaît sur les écrans
- **Complication** : Le signal s'arrête — comme s'il savait qu'ils étaient là
- **Crise** : Approcher ou maintenir la distance ?
- **Climax** : Vasquez ordonne une approche prudente
- **Résolution** : L'équipage contemple l'Objet en silence — sense of wonder

**Valeurs** :
- Externe : Théorie → Réalité
- Interne : Excitation → Émerveillement mêlé d'inquiétude

**Notes** : Premier vrai moment de sense of wonder. Description détaillée de l'Objet.

**~1 200 mots**

|---------|-------------|
| **Lieu** | Laboratoire du *Thulé* |
| **Temps** | 10 jours après l'arrivée |

### Scène 15 : Le Centre

**~1 500 mots — ALL IS LOST**

| Élément | Description |
|---------|-------------|
| **Lieu** | Centre de l'Objet |
| **Temps** | Jour 4 |
| **POV** | Élise |

**Résumé** : L'équipage atteint le centre de l'Objet. Ils découvrent des milliers de capsules, disposées en cercles concentriques. Chacune contient les restes préservés d'une espèce intelligente différente. L'Objet est un **mausolée** — un monument funéraire pour des civilisations mortes.

- **Incident déclencheur** : Arrivée au centre
- **Complication** : Les capsules révèlent leur contenu
- **Crise** : Accepter cette vérité ou la nier ?
- **Climax** : Élise comprend — c'est un cimetière galactique

**Valeurs** :
- Externe : Mystère → Révélation terrible
- Interne : Espoir → Point le plus bas

### Scène 16 : Le Message

**~1 300 mots — TURNING POINT ACTE 2**

| Élément | Description |
|---------|-------------|
| **Lieu** | Centre de l'Objet |
| **Temps** | Même jour |
| **POV** | Élise |

**Résumé** : Les hologrammes délivrent un message plus clair. Les créateurs ont parcouru la galaxie pendant des millions d'années. Ils n'ont trouvé que des espèces mortes ou mourantes. Ils ont créé l'Objet comme un héritage, placé là où un système solaire allait se former, en espérant que quelqu'un le trouverait. L'humanité est les **héritiers désignés**.

**TURNING POINT — RÉVÉLATEUR**

**Les Cinq Commandements** :
- **Incident déclencheur** : Nouveau cycle d'hologrammes
- **Complication** : Le message est fragmentaire mais compréhensible
- **Crise** : Que signifie être les "héritiers" ?
- **Climax** : Élise comprend — c'est un cadeau, pas un avertissement
- **Résolution** : Mais quel cadeau ? Quelque chose de plus attend.

**Valeurs** :
- Externe : Désespoir → Nouvelle perspective
- Interne : Nous ne sommes pas abandonnés — nous sommes choisis

### Scène 17 : L'Interface

**~1 200 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Centre de l'Objet |
| **Temps** | Jour 5 |
| **POV** | Élise |

**Résumé** : Élise découvre l'interface — une structure qui semble conçue pour une connexion neuronale. ARIA analyse : c'est un accès à la totalité de la connaissance des créateurs. Mais la connexion est risquée — interférence neuronale, risque de dommages ou de mort.

- **Climax** : Élise se porte volontaire

### Scène 18 : Le Choix Final

**~1 400 mots — CRISE ET CLIMAX ACTE 2**

| Élément | Description |
|---------|-------------|
| **Lieu** | Centre de l'Objet |
| **Temps** | Jour 5 |
| **POV** | Élise |

**Résumé** : Moment de décision. Vasquez interdit la connexion — trop dangereux. Élise doit choisir : obéir et rentrer avec ce qu'ils ont, ou risquer sa vie pour accéder à la connaissance. Elle pense à Léa. Elle pense à Marc. Elle pense à toutes les questions sans réponse de sa vie.

**CRISE ET CLIMAX DE L'ACTE 2**

**Les Cinq Commandements** :
- **Incident déclencheur** : Vasquez interdit la connexion
- **Complication** : Élise comprend que c'est maintenant ou jamais
- **Crise** : Obéir et rentrer, ou risquer tout ?
- **Climax** : Élise entre dans l'interface contre les ordres
- **Résolution** : La connexion commence. Blanc.

**Valeurs** :
- Externe : Décision → Action
- Interne : Point de transformation — Élise choisit la connaissance au prix de tout
