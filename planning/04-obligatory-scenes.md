# Conventions et Scènes du Hard SF

*"Les conventions sont les éléments que le lecteur s'attend à trouver. Les scènes obligatoires sont les événements qui DOIVENT se produire."*

## CONVENTIONS DU HARD SF

### 1. Le Novum
- [ ] **Défini dans votre histoire**

L'élément de nouveauté scientifique/technologique central.

**Types courants** :
- Découverte scientifique majeure
- Premier contact / artefact alien
- Technologie transformatrice
- Phénomène cosmique
- Évolution/transformation humaine

### 2. Le Protagoniste Compétent
- [ ] **Présent dans votre histoire**

Convention Heinlein par excellence :
- Intelligence et expertise
- Capacité à résoudre des problèmes
- Réaction rationnelle face à l'inconnu
- Pas nécessairement sympathique, mais admirable

**Domaine d'expertise** : _______________________________

---

### 3. Rigueur Scientifique
- [ ] **Maintenue dans votre histoire**

La science doit être :
- Basée sur des principes connus
- Ou extrapolée de manière logique
- Cohérente internement
- Jamais "magique" sans explication

**Domaines scientifiques impliqués** : _______________________________

---

### 4. Le Worldbuilding Cohérent
- [ ] **Établi dans votre histoire**

L'univers doit avoir :
- Des règles claires
- Des conséquences logiques
- Une technologie cohérente
- Une société plausible

---

### 5. Le Sense of Wonder
- [ ] **Présent dans votre histoire**

Moments où le lecteur ressent :
- L'immensité de l'univers
- Le sublime cosmique
- L'émerveillement scientifique
- La petitesse de l'humanité face au cosmos

### 6. La Dimension Philosophique
- [ ] **Intégrée dans votre histoire**

Le hard SF pose des questions :
- Sur la nature de l'humanité
- Sur notre place dans l'univers
- Sur les limites de la connaissance
- Sur les conséquences du progrès

## SCÈNES TYPIQUES DU HARD SF

Ces scènes ne sont pas toutes obligatoires, mais le lecteur s'attend à en trouver plusieurs.

### 1. La Scène de Découverte
- [ ] **Planifiée**

Le novum est révélé pour la première fois.

**Caractéristiques** :
- Moment d'étonnement ou de choc
- Le protagoniste réalise que tout change
- Souvent l'Incident Déclencheur

**Votre scène** : _______________________________

---

### 2. La Scène d'Investigation
- [ ] **Planifiée**

Le protagoniste cherche à comprendre le novum.

**Caractéristiques** :
- Méthode scientifique
- Hypothèses et tests
- Dialogue technique crédible

**Votre scène** : _______________________________

---

### 3. Le Problème Technique/Scientifique
- [ ] **Planifié**

Un obstacle qui nécessite l'intelligence pour être surmonté.

**Caractéristiques** :
- Pas de solution facile
- Exige réflexion et compétence
- Les enjeux sont réels

**Votre problème** : _______________________________

---

### 4. La Solution Ingénieuse
- [ ] **Planifiée**

Le protagoniste résout le problème de manière créative.

**Caractéristiques** :
- Logique et cohérente avec l'univers
- Utilise les connaissances établies
- Satisfaisante intellectuellement

**Votre solution** : _______________________________

---

### 5. La Révélation Cosmique
- [ ] **Planifiée**

Une compréhension plus large de l'univers ou du novum.

**Caractéristiques** :
- Change la perspective
- Implications profondes
- Souvent vers la fin

**Votre révélation** : _______________________________

---

### 6. Le Moment de Wonder (Clarke)
- [ ] **Planifié**

Pure sense of wonder — le sublime cosmique.

**Exemples** :
- Première vue d'un phénomène spatial
- Réalisation de l'échelle cosmique
- Contact avec l'ineffable

**Votre moment** : _______________________________

---

### 7. L'Infodump Élégant
- [ ] **Planifié**

Exposition technique nécessaire, rendue intéressante.

**Techniques** :
- Dialogue entre expert et novice
- Protagoniste qui découvre
- Narration intégrée à l'action
- JAMAIS de pavé didactique ennuyeux

---

### 8. La Scène de Compétence (Heinlein)
- [ ] **Planifiée**

Le protagoniste démontre son expertise.

**Caractéristiques** :
- Détails techniques crédibles
- Satisfaction de voir un expert travailler
- Établit la crédibilité du personnage

## TYPES DE STRUCTURES HARD SF

### Type A : L'Exploration (style Rama)

**Structure** :
1. Découverte d'un mystère
2. Investigation méthodique
3. Révélations progressives
4. Mystère partiellement résolu
5. Fin ouverte qui élargit le mystère

**Scènes clés** :
- [ ] Première rencontre avec le mystère
- [ ] Investigation scientifique
- [ ] Révélation partielle
- [ ] Fin qui pose plus de questions

---

### Type B : Le Problème (style The Martian)

**Structure** :
1. Situation de crise
2. Analyse du problème
3. Tentatives de solution
4. Complications
5. Solution finale ingénieuse

**Scènes clés** :
- [ ] Établissement du problème
- [ ] Tentative 1 (échec)
- [ ] Tentative 2 (échec partiel)
- [ ] Solution créative
- [ ] Résolution

---

### Type C : La Révélation (style 2001)

**Structure** :
1. Mystère initial
2. Progression vers l'inconnu
3. Transformation du protagoniste
4. Révélation transcendante
5. Nouveau paradigme

**Scènes clés** :
- [ ] Contact avec le mystère
- [ ] Voyage vers l'inconnu
- [ ] Point de non-retour
- [ ] Révélation/Transcendance
- [ ] Nouvelle existence

---

### Type D : La Société (style Heinlein social)

**Structure** :
1. Présentation d'une société future
2. Protagoniste en conflit avec le système
3. Exploration des implications
4. Climax social/politique
5. Résolution (changement ou non)

**Scènes clés** :
- [ ] Exposition du monde
- [ ] Friction protagoniste/société
- [ ] Point de rupture
- [ ] Confrontation
- [ ] Nouveau status quo

---

## CHECKLIST DE VALIDATION HARD SF

### Conventions fondamentales
- [ ] Le novum est défini et central
- [ ] Le protagoniste est compétent
- [ ] La science est rigoureuse
- [ ] Le worldbuilding est cohérent
- [ ] Il y a du sense of wonder
- [ ] Une dimension philosophique existe

### Scènes essentielles
- [ ] Scène de découverte
- [ ] Au moins un problème technique
- [ ] Au moins une solution ingénieuse
- [ ] Un moment de wonder
- [ ] Exposition intégrée (pas de pavé)

### Éviter les pièges
- [ ] Pas de "magic science" inexpliquée
- [ ] Pas de protagoniste stupide face à l'évidence
- [ ] Pas de deus ex machina
- [ ] Pas de technobabble vide de sens
- [ ] Pas de prêchi-prêcha sans subtilité
