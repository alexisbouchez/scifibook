## Les Cinq Commandements de l'Acte 1

### INCIDENT DÉCLENCHEUR GLOBAL

> **Élise détecte une anomalie gravitationnelle impossible dans les données de la sonde Horizon-7.** Les calculs révèlent un objet sphérique parfait de 12 km de diamètre dans la Ceinture de Kuiper, en orbite stable depuis plus longtemps que le système solaire existe.

**Type** : Coïncidence (Élise tombe sur les données par hasard en cherchant autre chose)

**Question dramatique** : Qu'est-ce que cet objet ? Qui l'a créé ? Pourquoi ?

### TURNING POINT DE L'ACTE 1

> **Révélateur** : L'analyse spectrographique révèle que le matériau de l'Objet n'existe pas dans le tableau périodique. C'est artificiel. C'est alien. La preuve définitive.

### CRISE DE L'ACTE 1

**Type** : Biens Irréconciliables

> Élise doit choisir entre **rester avec Léa** (sa fille de 16 ans qu'elle élève seule et avec qui elle a déjà une relation distante) **ou partir** pour une mission de 18 mois vers la plus grande découverte de l'histoire — peut-être sa dernière chance de donner un sens aux sacrifices de toute sa vie.

### CLIMAX DE L'ACTE 1

> **Élise choisit de partir.** Elle confie Léa à sa sœur. Dans la scène d'adieu, Léa ne pleure pas — elle est résignée. "Tu reviens toujours aux étoiles, maman."

### RÉSOLUTION DE L'ACTE 1

> Le *Thulé* quitte l'orbite terrestre. Huit mois de voyage commencent. Élise regarde la Terre rétrécir, consciente qu'elle a peut-être fait le mauvais choix — mais incapable de faire autrement.

## SCÈNES DÉTAILLÉES

### Scène 1 : La Découverte

**~1 200 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Observatoire de Paris, bureau d'Élise, nuit |
| **Temps** | Janvier 2147 |
| **POV** | Élise |

**Résumé** : Élise travaille tard, analysant des données de routine de la sonde Horizon-7. Elle cherche des preuves d'une théorie sur les objets transneptuniens. Au lieu de ça, elle trouve une anomalie gravitationnelle qui ne devrait pas exister.

**Les Cinq Commandements** :
- **Incident déclencheur** : Les données montrent une perturbation orbitale inexplicable
- **Complication** : Les calculs semblent impossibles — erreur instrumentale ?
- **Crise** : Ignorer l'anomalie ou passer la nuit à vérifier ?
- **Climax** : Élise vérifie. Les données sont solides.
- **Résolution** : Elle reste figée devant l'écran. "Ce n'est pas possible."

**Valeurs** :
- Externe : Ignorance → Premier indice de connaissance
- Interne : Routine → Excitation contenue

**Notes d'écriture** :
- Établir la compétence d'Élise (elle voit ce que d'autres manqueraient)
- Montrer son isolement (seule au bureau, tard)
- Ton : Précision technique, émerveillement retenu

---

### Scène 2 : La Vérification

**~1 000 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Même bureau + salle de calcul ESA |
| **Temps** | Jours suivants |
| **POV** | Élise |

**Résumé** : Élise passe plusieurs jours à vérifier ses calculs. Elle demande discrètement à des collègues de vérifier les données sans leur dire ce qu'elle cherche. Tout confirme : l'anomalie est réelle. Un objet de 12 km, parfaitement sphérique, en orbite stable dans la Ceinture de Kuiper.

**Les Cinq Commandements** :
- **Incident déclencheur** : Les vérifications confirment les données
- **Complication** : Un collègue sceptique remet en question la méthodologie
- **Crise** : Présenter la découverte maintenant (risquer sa réputation) ou attendre plus de preuves ?
- **Climax** : Élise décide d'attendre — elle veut des preuves irréfutables
- **Résolution** : Elle programme une observation directe via télescope spatial

**Valeurs** :
- Externe : Doute → Confirmation
- Interne : Certitude prudente

---

### Scène 3 : Le Signal

**~1 200 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Centre de contrôle ESA, Darmstadt |
| **Temps** | 3 semaines plus tard |
| **POV** | Élise |

**Résumé** : L'observation directe confirme l'existence de l'Objet. Mais plus important : les radioastronomes détectent un signal émanant de l'Objet. Une séquence de nombres premiers sur 1420 MHz — la fréquence de l'hydrogène, le "canal de l'eau", utilisé universellement pour la recherche de vie extraterrestre.

**Les Cinq Commandements** :
- **Incident déclencheur** : Détection du signal radio
- **Complication** : Le signal est trop régulier pour être naturel
- **Crise** : Publier la découverte (risque de panique/scepticisme) ou garder le secret ?
- **Climax** : Élise informe la direction de l'ESA en privé
- **Résolution** : Réunion d'urgence convoquée

**Valeurs** :
- Externe : Confirmation → Preuve d'intelligence
- Interne : Excitation → Gravité

**Notes d'écriture** :
- Moment de sense of wonder : la séquence de nombres premiers
- Référence implicite à Contact de Sagan
- Élise comprend l'ampleur de la découverte

---

### Scène 4 : La Preuve

**~1 200 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Siège ESA, Paris |
| **Temps** | 1 semaine plus tard |
| **POV** | Élise |

**Résumé** : Présentation devant le conseil scientifique de l'ESA. Scepticisme initial. Mais Élise présente l'analyse spectrographique : le matériau de l'Objet contient des éléments qui n'existent pas dans le tableau périodique. C'est la preuve irréfutable : l'Objet est artificiel et d'origine extraterrestre.

**TURNING POINT DE L'ACTE 1**

**Les Cinq Commandements** :
- **Incident déclencheur** : Élise présente ses données
- **Complication** : Scepticisme du conseil, questions agressives
- **Crise** : Révéler l'analyse spectrographique maintenant ou garder cette carte ?
- **Climax** : Élise révèle la preuve définitive — éléments inconnus
- **Résolution** : Silence dans la salle. Puis : "Nous devons envoyer une mission."

**Valeurs** :
- Externe : Scepticisme → Acceptation
- Interne : Élise seule contre tous → Validation

**Notes d'écriture** :
- Montrer la compétence d'Élise sous pression
- Le moment où tout change — plus de retour possible
- Établir les enjeux : c'est réel

---

### Scène 5 : Le Choix

**~1 500 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Appartement d'Élise, Paris / École de Léa |
| **Temps** | 2 mois plus tard |
| **POV** | Élise |

**Résumé** : La mission est approuvée. Élise est la scientifique en chef évidente — c'est sa découverte. Mais cela signifie 18 mois loin de Léa. Scène avec Léa qui révèle la tension entre elles. Léa est résignée, presque froide. Élise comprend qu'elle a déjà perdu sa fille d'une certaine façon.

**CRISE ET CLIMAX DE L'ACTE 1**

**Les Cinq Commandements** :
- **Incident déclencheur** : Élise reçoit la confirmation : elle est sélectionnée
- **Complication** : Léa apprend la nouvelle et réagit avec froideur
- **Crise** : Rester avec Léa ou partir pour la mission ?
- **Climax** : Élise choisit de partir
- **Résolution** : Conversation difficile avec Léa. "Tu reviens toujours aux étoiles, maman."

**Valeurs** :
- Externe : Opportunité → Décision
- Interne : Culpabilité → Choix assumé (mais douloureux)

**Notes d'écriture** :
- Scène émotionnellement la plus importante de l'Acte 1
- Montrer la faille d'Élise — son incapacité à connecter
- Établir l'arc émotionnel à résoudre

---

### Scène 6 : Le Départ

**~1 400 mots**

| Élément | Description |
|---------|-------------|
| **Lieu** | Base de lancement / Vaisseau *Thulé* |
| **Temps** | Avril 2147 |
| **POV** | Élise |

**Résumé** : Jour du départ. Brève introduction de l'équipage (Vasquez, Okonkwo, Kowalski). Le *Thulé* quitte l'orbite terrestre. Élise regarde la Terre rétrécir. Elle pense à Léa. Huit mois de voyage commencent.

**RÉSOLUTION DE L'ACTE 1**

**Les Cinq Commandements** :
- **Incident déclencheur** : Compte à rebours, lancement
- **Complication** : Dernière hésitation d'Élise
- **Crise** : Trop tard pour revenir en arrière
- **Climax** : Le *Thulé* quitte l'orbite — point de non-retour
- **Résolution** : Élise seule devant le hublot, la Terre devient un point bleu

**Valeurs** :
- Externe : Terre → Espace / Connu → Inconnu
- Interne : Doute → Engagement

**Notes d'écriture** :
- Introduction efficace de l'équipage (pas de longues descriptions)
- Moment de solitude d'Élise
- Fin de l'Acte 1 : vers l'inconnu
