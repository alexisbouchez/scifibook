# Scene Spreadsheet / Grille de Scènes

## *L'Anomalie de Thulé*

---

## Légende

### Polarity Shift (Changement de Polarité)
- **+** = Valeur positive
- **-** = Valeur négative
- **++** = Très positif
- **--** = Très négatif

### Types de Turning Point
- **A** = Actif (action)
- **R** = Révélateur (information)

### Types de Crise
- **MMC** = Meilleur Mauvais Choix
- **BI** = Biens Irréconciliables
